# Godot-generic-bullet-hell-game

Hi, this is the second game i ever made. Top down, pixel-art, wave based, bullet hell game (as you can guess).
It has 2 guns, 3 enemies, upgrages, items, parrying and a custom level editor.
Your goal is to reach wave 30.

The game and the source code is in itch.io page: https://turkenx.itch.io/generic-bullet-hell-game

The games showcase https://youtu.be/ZERNeHbhlYk
